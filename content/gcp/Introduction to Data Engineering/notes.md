
## Data engineering tasks
1. ### Replicate and Migrate
    * onboard the date into google cloud.
2. ### Ingest
    * data is converted into data source that can be further transformed into valuable insights. Sample data sources are cloud storage,Pub/Sub, Asynchronoud messaging service.
3. ### Transfor 
    * transformation services add new value to data,(dataproc,dataflow,dataform)
4. ### Store 
    * Data sinks stores the processed and transformed data and is readily avaliable for analytical purpose. 
    * refer to day2 notes in common folder for different types of data sinks available




### Big Query:
* used to store structured or semistructured data mainly for OLAP
* serverless datawarehouse tool, 
* bigquery organizes datatables as units called datasets, wevan refer to a table in a query using ```project.dataset.table```
