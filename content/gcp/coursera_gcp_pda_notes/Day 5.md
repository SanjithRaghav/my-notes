### Dataproc

#### Limitations of on-prem hadoop:
* No separation between storage and compute resources
* Hard to scale fast
* capacity limit.

#### Dataproc for hadoop
* built in support for hadoop
* managed hardware and software configuration
* simplified version management
* flexible job configuration.
#### running Hadoop on dataproc
* low cost(second by second billing, with 1 minute minimum billing period)
* super fast
* resizable clusters
* open source ecosystem
* integrated with other cloud services like bigquery, bigtable etc.
* fully managed
* versioning
* highly available
* developer tools

#### other operations suport systems available are 
* optional components, opt it while deploying from the console.
* initialization action, follow the steps given in docs to configure these services in the cluster.
